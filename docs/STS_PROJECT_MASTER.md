# STS Project Master

## Current State
The live website is a temporary speed-launch version inside WordPress.
It uses custom HTML, CSS, and JS with app-like transitions.

## Locked Strategic Decision
The final website will be rebuilt as a code-first, real multi-page site.
WordPress remains the CMS/admin shell, not the primary design engine.

## Visual System
Preserve:
- dark premium technical look
- blue/cyan accents
- glass UI
- terminal-inspired elements
- blue light page transitions

Avoid:
- generic SaaS look
- soft startup visuals
- Elementor-template aesthetics

## Core Pages
- Homepage
- Lösungen
- Preise
- Über uns
- Kontakt
- FAQ
- legal pages

## Current Priority
Stabilize live version enough for public presence.
Then rebuild properly in code.

## Long-Term Goal
Turn the STS website into:
1. company website
2. portfolio flagship
3. reusable client-site framework
