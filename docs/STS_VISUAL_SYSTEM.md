# STS Visual System

## Identity
Premium technical execution layer.
Dark, serious, structured, controlled.

## Core Visual Traits
- deep navy / black base
- cyan and electric-blue accents
- glass surfaces
- controlled glow
- precise spacing
- premium contrast
- technical atmosphere

## Interaction Traits
- smooth transitions
- blue light overlay between pages
- subtle hover depth
- terminal-inspired motion
- no gimmicky animation overload

## Must Avoid
- generic agency look
- generic SaaS landing-page look
- compliance cliche visuals
- warm beige or soft pastel systems
- blob shapes
